创建

创建私有源 git 仓库，可以创建 GitHub 仓库或者其它，然后将私有源添加至本地，下以 git@github.com:Coder-ZJQ/Specs.git 为例：

# pod repo add NAME URL [BRANCH]
pod repo add zjq-specs git@github.com:Coder-ZJQ/Specs.git

可以执行下列命令查看已添加的私有源：

# 查看本地的私有源列表
pod repo list

校验私有源中的 spec 是否合法

cd ~/.cocoapods/zjq-specs
pod repo lint .
