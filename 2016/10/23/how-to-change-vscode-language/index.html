vscode 更改显示语言

发表于 2016-10-23 分类于 工具使用 ， VSCode 阅读次数： 本文字数： 426 阅读时长 ≈ 1 分钟

Configure Language 指令

  1. shift + command 唤出 Command Palette
  2. 输入 Configure Language 确定
  3. 编辑 locale.json 文件
    {
    // 定义 VSCode 的显示语言。
    // 请参阅 https://go.microsoft.com/fwlink/?LinkId=761051，了解支持的语言列表。
    // 要更改值需要重启 VSCode。
    "locale":"en"
}
